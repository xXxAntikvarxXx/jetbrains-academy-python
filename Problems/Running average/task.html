Running average Open on JetBrains Academy

Read a string with digits from the input and convert each number to an integer. For your newly created list, calculate the running average according to the following formula:

\[A_i = \frac{x_i + x_{i+1}}{2},\] where \(x_i\) and \(x_{i+1}\) are elements of the original list, \(A_i\) is the element of the moving average.

Print the result. Notice that this list will have one less item.


Sample Input:
12345
Sample Output:
[1.5, 2.5, 3.5, 4.5]


Memory limit: 256 MB
Time limit: 15 seconds
