We have designed a function that tells you what recipe you can make based on the ingredient you have.

The function special takes your ingredient, e.g. "cinnamon", and several dishes as keyword arguments, e.g. apple pie=["apple", "sugar", "cinnamon", "flour", "egg", "butter"], omelet=["egg", "milk", "bacon", "salt", "pepper"], to print potential dishes of the day.

The needed variables are defined in the program:

  1. The variable ingredient contains the dish you want to cook.
  2. The variable dishes contains a dictionary where the keys are the dishes, and the values are the ingredients for these dishes.

Your task is to call the function and pass the mentioned variables as arguments. The dishes dictionary should be unpacked, and the key-value pairs should be passed as keyword arguments.

Note that you should NOT handle the input, since dishes, ingredient and the function special are already defined.

Mind the order: first, positional arguments, and then the rest. You cannot pass the dictionary without unpacking. What should we do to pass on keyword arguments?
