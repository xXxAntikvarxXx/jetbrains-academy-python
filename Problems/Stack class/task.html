Implement your own Stack class in Python.

It should wrap some built-in Python data structure and implement the following methods:

  • push(el): push a new element el to the stack
  • pop(): remove and return the last element from the stack
  • peek(): return the last element without removing it
  • is_empty(): check if the stack is empty and return True or False

Upon initialization, the stack must be empty. You do NOT need to create an instance of the class or print anything.

The following sequence of operations is implemented on your class in order to test your code: 

is_empty()
push(0)
push(1)
push(2)
push(3)
push(4)
pop()
pop()
peek()
pop()
is_empty()
