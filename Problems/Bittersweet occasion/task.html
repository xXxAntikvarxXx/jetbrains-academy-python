Bittersweet occasion

Mysterious things have happened on the Hogwarts Express! Someone enchanted a food trolley and all the goodies got mixed up on the shelves. While the trolley witch is looking for the culprit, you will place Drinks, Pastry and Sweets on three respective shelves.

These base classes have various children, for example, PumpkinJuice inherits from Drinks. Write a function that finds a parent class for an unknown class stored in the variable child and prints out the parent's name.

Note: all these classes are already created, you don't need to do that. You only need to work with the child.

You can use the issubclass() function to check is a particular class is the parent class.


Sample Input:
PumpkinJuice

Sample Output:
Drinks


Sample Input:
CauldronCake

Sample Output:
Pastry


Sample Input:
ChocolateFrog

Sample Output:
Sweets

Time limit: 15 seconds
