Let's say you need to keep track of the arguments passed to the script test.py. You know that it takes four arguments: write a module that gets them from the command line and prints them as a list. All arguments will be numerical.

The interaction with the command line takes place in the computer itself so we can't check this task directly. So let's make it this way: the input will be the list args containing what you would otherwise get from sys.argv, and you should just write what you would do next.

E.g. args = ["script.py", "1", "2", "3", "4"], then your output should be [1, 2, 3, 4] with the list elements as numbers.
